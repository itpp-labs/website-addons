Stop Online Sales

Adds options to disable all sales and hide all prices, but keep products visible at website.

Key features:
  • hide product prices in your website shop
  • hide an ability to add a product to a shop cart
  • use the website shop for demonstration, but avoiding shopping

The module adds new items to the "Customize" menu of you website. In order to see this menu you have to sign in to the website as a user with the website "Editor and Designer" right.
