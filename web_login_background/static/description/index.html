Web Login Background

Get a random background at the login page.

Usage

After installing the module, yon should go to Settings and create an attachment (background image)

Select an image, put a checkmark in the "Use as login page background" checkbox and click on Save button

Now, turning to the login page, you can see the downloaded background

The module allows to use unlimited number of images for the background. In this case, an image will be selected and changed randomly. For example, if we reload the above login page, the background will be changed.

It must be noted that a background images can be adapted to any size screen.
